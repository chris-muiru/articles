## HTTP Headers
key:value pairs that provide additional data to browser to help complete the request.
- General headers
- Request headers
```
General Headers
  Request URL: https://www.google.com/
  Request Method: GET
  Status Code: 200
  
Request Headers
    Accept: text/html
    Accept-Language: en-US
    Connection: keep-alive
```
